# API SMM — Venda de Serviços (likes, seguidores, etc.)

API para gerenciamento e venda de serviços SMM. Implementada em TypeScript com NestJS e Prisma, esta API expõe endpoints para criação de ordens, gerenciamento de serviços, usuários e carteiras, além de integração via webhook.

## Principais funcionalidades

- Criação e gerenciamento de ordens de compra de serviços.
- Cadastro e gestão de serviços oferecidos.
- Controle de usuários e carteiras (saldo/internal ledger).
- Painel de filas/processamento assíncrono para execução de tarefas.

## Stack Tecnológica

- Node.js + NestJS
- TypeScript
- Prisma + PostgreSQL
- Redis (para filas/cache)
